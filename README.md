# Sonic Boilerplate

A simple boilerplate to create projects with Gulp, Stylus, Jeet and Rupture.

## Getting Started

### Instalation

```
# Clone this repository
$ git clone git@github.com:Stoque/Sonic.git new_project
$ cd new_project

# install gulp globally
$ npm install -g gulp

# install dependencies
$ npm install
```
